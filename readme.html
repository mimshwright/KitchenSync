KitchenSync Library

Setup

To set this library up for editing in FlexBuilder:

  1. Create a new Flex Library Project
  2. Set the project name to KitchenSyncLib
  3. Uncheck use default location and enter the path to the SVN working copy you checked out
  4. Click Next
  5. Add src as your main source folder at the bottom.
  6. Click Select All.
  7. Click the Library Path tab
  8. Click add SWC Folder and enter libs/as3lib/bin
  9. Click finish
