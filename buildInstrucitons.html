Building KitchenSync

This file shows instructions on how to set up a project to build the KitchenSync library.

Creating a pre-compiled SWC library

Creating a Flex Library Project in FlexBuilder / FlashBuilder

  1. In FlashBuilder, do File -> New -> Flex Library Project
  2. For project name, use 'kitchensynclib'
  3. For location, make sure to point to the root folder of the kitchensync project. Click next.
  4. In the Build Path settings, keep all default settings and click the "Add SWC Folder" button. Use "libs" as the path. Under classes, make sure all the classes are checked (FB4 users, click the "Include all classes from all source paths").
  5. The resulting SWC will appear in the bin folder.

Using Apache Ant

If you are familiar with the ant build tool and the Flex SDK, this is the best method for creating a SWC. Use the following steps:

  1. Create a duplicate of build.properties.template and name it build.properties. In this file, you should adjust the values to point to the location of the Flex SDK on your machine.
  2. If you don't have a 'bin' folder in the project, create one now.
  3. Run the "build" target of the ant script. ant build The resulting SWC will appear in the bin folder.
