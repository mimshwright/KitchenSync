Classpublic class KitchenSyncDefaults

Since : 1.1

All default settings for KitchenSync can be configured through this class. You can change the values by setting the public vars before initializing KitchenSync or by changing this class directly in a local copy of the library.

public static var autoDelete:Boolean = false

Determines whether actions will attemp to delete themselves when they complete. Set this to true if you are having memory or performance issues with your app.

snapToIntegerproperty 
public static var snapToInteger:Boolean = false

When set to true, the values that a tween generates will always be rounded to the nearest integer. This can make the animated image appear sharper but can make slower animations appear a little jumpy. Use this when you need text to appear 'on-pixel'.

The default value is false.

snapToValueOnCompleteproperty 
public static var snapToValueOnComplete:Boolean = true

When set to true, the value at the end of a tween will always be set to the endValue regardless of what the easing function interpoplates.

The default value is true.
