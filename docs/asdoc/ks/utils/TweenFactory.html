newFadeInTween()method
public static function newFadeInTween(target:DisplayObject, duration:* = 0, delay:* = 0, setVisibility:Boolean = true, startFromZeroAlpha:Boolean = false):IAction

Creates an alpha tween that fades in. The list of parameters are simplified to keep this function short and practical.

Parameters

target:DisplayObject — Tween's target must be a display object.
 
duration:* (default = 0) — The time in milliseconds that this tween will take to execute. String values are acceptable too.
 
delay:* (default = 0) — The time to wait in milliseconds before starting the tween. String values are acceptable too.
 
setVisibility:Boolean (default = true) — If true, this sets the visibility to true before the fade in is complete.
 
startFromZeroAlpha:Boolean (default = false) — If true will set the alpha to 0 before executing the fade in.

Returns

newScaleTweenAdvanced()method 
public static function newScaleTweenAdvanced(target:DisplayObject, startValueX:Number, endValueX:Number, startValueY:Number, endValueY:Number, centerPointX:Number = 0, centerPointY:Number = 0, duration:* = 0, delay:* = 0, easingFunciton:Function = null, easingMod1:Number, easingMod2:Number):KSTween

Since : 2.0.1

Creates a tween that scales x and y but with more options than newScaleTween(). Note that AUTO_TWEEN_VALUE won't work correctly with this tween because it determines the x and y positioning at the time that it is created.

Parameters

target:DisplayObject — The display object whose property will be changed
 
startValueX:Number — The starting scaleX of the target.
 
endValueX:Number — The ending scaleX of the target.
 
startValueY:Number — The starting scaleY of the target.
 
endValueY:Number — The ending scaleY of the target.
 
centerPointX:Number (default = 0) — The point in the displayObject's space that will be the centerpoint of the scale. Use a percentage rather than a pixel number. Default is 0, or left.
 
centerPointY:Number (default = 0) — The point in the displayObject's space that will be the centerpoint of the scale. Use a percentage rather than a pixel number. Default is 0, or top.
 
duration:* (default = 0) — The time in milliseconds that this tween will take to execute. String values are acceptable too.
 
delay:* (default = 0) — The time to wait in milliseconds before starting the tween. String values are acceptable too.
 
easingFunciton:Function (default = null) — The function to use to interpolate the values between fromValue and toValue.
 
easingMod1:Number (default = NaN) — An optional modifier to the easing funtion.
 
easingMod2:Number (default = NaN) — An optional modifier to the easing funtion.

Returns
KSTween — A new KSTween instance.

Example 
         // scale sprite from 0 to 100% from the centre of the sprite over 1 second, no delay, with Cubic ease-in-out. 
         TweenFactory.newScaleTweenAdvanced(
             sprite, 
             0, 1, 
             0, 1,  
             0.5, 0.5, 
             1000, 0, Cubic.easeInOut)

public static function newTween(targets:*, properties:*, startValue:Number, endValue:Number, duration:* = 0, delay:* = 0, easingFunction:Function = null, easingMod1:Number, easingMod2:Number):KSTween

Creates a property tween with one or more targets.

Parameters

targets:* — The object whose property will be changed or an array of targets.
 
properties:* — The name of the property to change or an array of properties to change. The property must be a Number, int or uint such as a Sprite object's "alpha"
 
startValue:Number (default = NaN) — The starting value of the tween. By default, this is the value of the property before the tween begins.
 
endValue:Number (default = NaN) — The value to tween the property to. After the tween is done, this will be the value of the property.
 
duration:* (default = 0) — The time in milliseconds that this tween will take to execute. String values are acceptable too.
 
delay:* (default = 0) — The time to wait in milliseconds before starting the tween. String values are acceptable too.
 
easingFunction:Function (default = null) — The function to use to interpolate the values between fromValue and toValue.
 
easingMod1:Number (default = NaN) — An optional modifier to the easing funtion.
