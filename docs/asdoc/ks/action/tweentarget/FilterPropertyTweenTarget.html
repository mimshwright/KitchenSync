Classpublic class FilterPropertyTweenTarget
InheritanceFilterPropertyTweenTarget Object
Implements IFilterTweenTarget

Since : 1.5

A tween target that targets a property of a BitmapFilter object. For example, you may target the "blurX" property of a BlurFilter on a particular movieClip. Tweens will be applied to the filter class rather than the individual filter objects because in order for the filter to be applied to the display object, the array of filters must be rewritten on each update.
