ks.action Interfaces  | Classes
 


Interfaces
 InterfaceDescription
 IAction An action is a type that represents any kind of happening that takes place at a certain time.
 IActionGroup A group of other actions that is itself an action.
 IJumpableAction An interface that adds methods to IAction for jumping to a certain time or by a certain time.
 ILoaderAction A type of action that loads data from some external source when the start() mehtod is called and is considered finished when the transfer of data is complete.
 IPrecisionAction Used in certain action classes for precision timing.
 ITween A special type of action that changes a numeric value (such as the x position of a Sprite) over time.
Classes
 ClassDescription
 AbstractAction A default implementation of IAction.
 AbstractActionGroup A default implementation of IActionGroup.
 KSActionController An action that controls another action when executed.
 KSAddChild Same as calling addChildAt with a KSFunction.
 KSAsyncFunction An action for calling an asynchronous function that is expected to dipatch an event when it completes.
 KSAsyncLoop Allows you to split up a loop (like a for or while loop) into smaller chunks that execute over a period of time (rather than all at once).
 KSDispatchEvent An object that causes an event to be dispatched at the time of execution.
 KSFunction Packages up a function to be run at a specified time or as part of a sequence.
 KSLoader An action for loading swf or image files.
 KSLoadQueue Allows you to very quickly create a loading queue to load multiple files in a batch.
 KSLooper A decorator-like group for looping an action a set number of times.
 KSNullAction An empty action that doesn't do anything and has no duration.
 KSParallelGroup A group of actions that all start simultaneously when the group's start() method is called.
 KSRandomGroup A group of actions based on the sequence group that plays the child actions back in a random order.
 KSRemoveChild Same as using a KSFunciton to call removeChild.
 KSSequenceGroup A group of actions that execute one at a time in the order that they were added to the group.
 KSSetProperty An action for setting a property for an object.
 KSSimpleTween A simplified version of the KSTween class with NO bells or whistles.
 KSSimultaneousEndGroup A simultaneous-end-group or reverse-parallel-group where where all children END at the same time instead of starting at the same time.
 KSSound A sound that will be played when the action executes.
 KSStaggeredGroup A group of actions that executes in sequence seperated by a specified gap.
 KSTrace An action for tracing a message.
 KSTween Used for animating an object's properties, such as position or scale, over time.
 KSURLLoader A KitchenSync action version of a URLLoader.
 KSWait Does nothing except wait for the delay to pass.
 KSXMLLoader A KitchenSync aciton to load an XML file.
