Classpublic final class KitchenSync

Gateway to the library. Initializes the entire system.


Example
Initialize with default settings. 
   KitchenSync.initialize();
Initialize with a specific core and version checking. 
   KitchenSync.initializeWitCore(new EnterFrameCore(this), "2.0");

Method detail
initialize()method
public static function initialize(frameRate:int = 30, versionCheck:String):void

Initializes the timing core for KitchenSync. Must be called before using any actions.

Parameters
frameRate:int (default = 30) — Is an optional parameter that allows you to set the rate of the dispatches. This can be different than the SWF's framerate but for best results, use the framerate of the swf. The system may not perform well over 60.
 
versionCheck:String — a string for the version you think you're using. e.g. 1.2 This is recommended but not required. It will throw an error if you're using the wrong version of KS.
initializeWithCore()method 
public static function initializeWithCore(synchronizerCore:ISynchronizerCore, versionCheck:String):void

Allows you to specify a synchronizer core when you initialize the system. This can be any of the ISynchronizerCores and will allow you to use enterframes, timers, or other mehtods to drive the dispatches of your synchronizer. If this sounds too complicated, you can use the default settings by calling KitchenSync.initialize()

Parameters
synchronizerCore:ISynchronizerCore — The synchronizer core to use to drive updates.
 
versionCheck:String — a string for the version you think you're using. e.g. 1.2 This is recommended but not required. It will throw an error if you're using the wrong version of KS.
