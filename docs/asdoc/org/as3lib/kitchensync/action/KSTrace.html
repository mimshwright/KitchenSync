InheritanceKSTrace KSFunction AbstractAction flash.events.EventDispatcher

Since : 0.1

An action for tracing a message. Very useful when stringing together several actions in a group to report the progress of a group but can also be used to simply delay a trace by a specified time.

Examples
     // trace a message after 5 seconds
     new KSTrace("5 seconds have elapsed.", 5000).start();
     
     // trace when the first url in the group finishes loading.
     new KSSequenceGroup(
         new KSURLLoader("http://www.example.com/1"),
         new KSTrace("Example 1 loaded. loading example 2."),
         new KSURLLoader("http://www.example.com/2")
     ).start();
