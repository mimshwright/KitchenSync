Classpublic class KSNullAction
InheritanceKSNullAction KSWait AbstractAction flash.events.EventDispatcher

Since : 2.0.1

An empty action that doesn't do anything and has no duration. This can be used, for example, when you need a function to generate an action but you don't want the action to do anything.
