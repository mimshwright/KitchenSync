InheritanceKSAsynchronousFunction KSFunction AbstractAction flash.events.EventDispatcher
SubclassesKSLoader, KSURLLoader

An action for calling an asynchronous function that is expected to dipatch an event when it completes. This could be, for example, a server-side operation such as loading a file from a remote system.
