Classpublic class SaturationTweenTarget
InheritanceSaturationTweenTarget AbstractColorMatrixTweenTarget Object

Since : 1.6

A color matrix tween that affects saturation. The range of values should be between -1 (greyscale) and 1 (full saturation) with a value of 0 being no adjustment to the saturation.
