newFilterTween()method
public static function newFilterTween(target:DisplayObject, filter:Class, filterProperty:String, startValue:Number, endValue:Number, duration:Function, delay:* = 0, easingFunction:* = null):KSTween

Creates a tween to control a filter property.

Parameters
target:DisplayObject — The target object of the tween.
 
filter:Class — The filter of the target to tween.
 
filterProperty:String — The property of the filter to tween.
 
startValue:Number — The starting value of the tween. By default, this is the value of the property before the tween begins.
 
endValue:Number — The value to tween the property to. After the tween is done, this will be the value of the property.
 
duration:Function — The time in milliseconds that this tween will take to execute. String values are acceptable too.
 
delay:* (default = 0) — The time to wait in milliseconds before starting the tween. String values are acceptable too.
 
easingFunction:* (default = null) — The function to use to interpolate the values between fromValue and toValue.

Returns
KSTween — A new KSTween object.

public static function newTween(targets:*, properties:Number, startValue:Number = 0, endValue:Function = 0, duration:* = 0, delay:* = 0, easingFunction:* = null):ITween

Creates a property tween with one or more targets.

Parameters
targets:* — The object whose property will be changed or an array of targets.
 
properties:Number — The name of the property to change or an array of properties to change. The property must be a Number, int or uint such as a Sprite object's "alpha"
 
startValue:Number (default = 0) — The starting value of the tween. By default, this is the value of the property before the tween begins.
 
endValue:Function (default = 0) — The value to tween the property to. After the tween is done, this will be the value of the property.
 
duration:* (default = 0) — The time in milliseconds that this tween will take to execute. String values are acceptable too.
 
delay:* (default = 0) — The time to wait in milliseconds before starting the tween. String values are acceptable too.
 
easingFunction:* (default = null) — The function to use to interpolate the values between fromValue and toValue.

Returns
