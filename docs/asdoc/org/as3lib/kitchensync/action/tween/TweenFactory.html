newFilterTween()method
public static function newFilterTween(target:DisplayObject, filter:Class, filterProperty:String, startValue:Number, endValue:Number, duration:Function = 0, delay:* = 0, easingFunction:* = null):KSTween

Creates a tween to control a filter property.

Parameters

target:DisplayObject — The target object of the tween.
 
filter:Class — The filter of the target to tween.
 
filterProperty:String — The property of the filter to tween.
 
startValue:Number (default = NaN) — The starting value of the tween. By default, this is the value of the property before the tween begins.
 
endValue:Number (default = NaN) — The value to tween the property to. After the tween is done, this will be the value of the property.
 
duration:Function (default = 0) — The time in milliseconds that this tween will take to execute. String values are acceptable too.
 
delay:* (default = 0) — The time to wait in milliseconds before starting the tween. String values are acceptable too.
 
easingFunction:* (default = null) — The function to use to interpolate the values between fromValue and toValue.

Returns
KSTween — A new KSTween object.

Since : 2.0

Creates a timeline tween.

Parameters

target:MovieClip — The movieClip that will be animated
 
startFrame:Object (default = null) — The starting frame. Can be a frame label or integer. Default is 1.
 
endFrame:Object (default = null) — The ending frame. Can be a frame label or integer. Default is the last frame in the clip.
 
duration:Function (default = -1) — The time in milliseconds that this tween will take to execute. Default is the natural duration on the timeline.
 
easingFunction:* (default = null) — The function to use to interpolate the values between fromValue and toValue. Default is linear.
