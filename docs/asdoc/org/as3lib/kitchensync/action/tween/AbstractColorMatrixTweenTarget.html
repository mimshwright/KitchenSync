Classpublic class AbstractColorMatrixTweenTarget
InheritanceAbstractColorMatrixTweenTarget Object
Implements IFilterTweenTarget
Subclasses BrightnessTweenTarget, ContrastTweenTarget, SaturationTweenTarget

Since : 1.6

A type of tween target that uses a ColorMatrixFilter (using the ColorMatrix object from Grant Skinner). This class is used as a base for other tween targets that adjust individual properties such as saturation or brightness.

See also

http://www.gskinner.com/blog/archives/2007/12/colormatrix_upd.html
