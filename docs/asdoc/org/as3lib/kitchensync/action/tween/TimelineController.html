Packageorg.as3lib.kitchensync.action.tween
Classpublic class TimelineController
ImplementsITweenTarget

A TweenTarget that controls a timeline animation in a MovieClip. It allows you to animate between two frames in a MovieClip using a KSTween so that you can go forwards, backwards, change the duration and use easing functions on a timeline animation.

See also
