Classpublic class SoundTransformTarget
InheritanceSoundTransformTarget Object
Implements ITweenTarget

Since : 1.6

A tween target for setting sound properties of a SoundTransform object (e.g. 'volume'). You will provide an object that has a soundTransform property and a property to change. known issue: high speed transitions, such as oscicators with high frequencies, can cause clicks and distortion in the sound.

Examples
     
     var sound:Sound;
     // load your sound here...
     
     // fade up over 5 seconds
     var fadeUp:ITweenTarget = new SoundTransformTarget(sound, SoundTransformProperty.VOLUME, 0, 1);
     var tween:KSTween = TweenFactory.newTweenWithTargets(fadeUp, "5s");
     tween.start();
