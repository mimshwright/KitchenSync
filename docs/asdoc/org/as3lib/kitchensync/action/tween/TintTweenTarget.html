_tintOpacityproperty 
protected var _tintOpacity:Number = 1.0

A modifier (between 0 and 1) to the tint color that affects its intensity. For example, a tint of 0xFF0000 and a tintAmount of 0.5 would appear as a red tint with 50% opacity. This only affects the endValue. The default value is 1.0 or 100% opacity.

Constructor detail
TintTweenTarget()constructor
public function TintTweenTarget(target:DisplayObject, startValue:Number, endValue:*, tintOpacity:* = 1)

Constructor.

Parameters
target:DisplayObject — The display object to affect.
 
startValue:Number — The color to begin on. This can be a color hex value (uint), a ColorTransform object, or one of the special values CURRENT_TINT or NO_TINT. null values will also be interpreted as CURRENT_TINT.
 
endValue:* — The color to tint to. See notes for startValue.
 
tintOpacity:* (default = 1) — A modifier for the opacity of the endValue tint color. Ignored if endValue is a ColorTransform.
