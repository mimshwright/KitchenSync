Interfacepublic interface ISynchronizerCore
ImplementorsEnterFrameCore, SetIntervalCore, TimerCore

Provides a timing core for the Synchronizer. This allows the user to swap out different methods for time-keeping, be it an enterFrame based system, a Timer, setInterval, or some other event. Each pulse from the ISynchronizerCore will trigger the dispatchUpdate() event in the Synchronizer.
