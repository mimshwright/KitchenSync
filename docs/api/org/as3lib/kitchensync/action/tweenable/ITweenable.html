Packageorg.as3lib.kitchensync.action.tweenable
Interfacepublic interface ITweenable
SubinterfacesIFilterTweenable
ImplementorsTargetProperty, TimelineController

A bundle that wraps up a property or other value that can be tweened by a KSTween. Generally speaking, a KSTween will handle the timing of the tween while the ITweenable handles the values of the tween.
