Packageorg.as3lib.kitchensync.utils
Classpublic class FrameRateUtil
ImplementsISynchronizerClient

A singleton class that can track the current and average framerate of the KitchenSync system (when using EnterFrameCore, this is the framerate of the movie itself). The number of frames included in teh average framerate can be set as well. For a quick visual display of the framerate, use FrameRateView.
