_snapToValueOnCompleteproperty 
protected var _snapToValueOnComplete:Boolean = true

Indicates whether the final value for the easing function should snap to the target _toValue. If set to true, the target property will equal _toValue regardless of the results of the easing function.

The default value is true.

_snapToWholeNumberproperty 
protected var _snapToWholeNumber:Boolean = false

Indicates whether tweened values should snap to whole value numbers or use decimals. If set to true, the results of the easing functions on the target property will be rounded to the nearest integer.

The default value is false.

public function Tween(target:Object, property:String, toValue:Number, fromValue:Number, duration:int = 0, offset:int = 0, easingFunction:Function = null)

Constructor -todo - make all of these optional

Parameters
target:Object — - the object whose property will be changed.
 
property:String — - the name of the property to change. Must be a numeric property such as a Sprite object's "alpha"
 
toValue:Number — - the target value to tween the property to.
 
fromValue:Number — - the starting value of the tween. By default, this is the value of the property before the tween.
 
duration:int (default = 0) — - the time in frames that this tween will take to execute.
 
offset:int (default = 0) — - the time to wait before starting the tween.
 
easingFunction:Function (default = null) — - the function to use to interpolate the values between fromValue and toValue.
